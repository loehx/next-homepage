# Say 

## Usage

Run the `say` command with a brief, clear message:

```bash
say "Hey Alex, I need you!"
```

## Message Examples

| Scenario | Command |
|----------|---------|
| Need input | `say "Hey Alex, I need your input"` |
| Browser help | `say "Hey Alex, Please help with the browser"` |
| Done | `say "Hey Alex, all done. Ready to commit."` |
| Status | `say "Hey Alex, changes are done. Continuing with the review."` |
